Scrabble for supybot.

##Author

Nicolas Paris
http://www.nicosphere.net

##Features

- 378989 words dictionnary
- find the best word
- keep score in plain text file

##Commands

- !start
- !stop
- !next
- !what

##Configuration

So far, it's in the plugin.py file
- CHANNEL: channel it's should respond
- MAX_LETTERS: number of letter in each hand
- LAPS_TIMES: timeout for answering
